								흠시
								2018. 8. 17. 14:10
								
							
						
						
							당신이 컴퓨터공학과 출신이거나 혹은 신입 개발자 취업 준비를 해왔다면, 수 많은 개발자 분들의 블로그를 보았을 것이다.나 역시 그러했다. 세상엔 참 천재같으면서도 친절한 사람이 많더라. 알고싶은 것들에 대해 키워드만 넣어 구글링하면, 이미 공부하신 다른 분들의 지식, 그들의 고민과 노력에 대해서 알 수 있게 된다. 나는 단지 후발주자로서 많은 혜택 받고 산다.
이러한 생각과 느낌은 비단 나만 그런 것은 아닐 것이다. 처음에 한 두번은, 그냥 간단히 검색겸 혹은 전공 수업에서 이해 안되는 것들에 대해서 하나씩 검색해보다가, 어느 순간 이런 지식의 공유는 개발자들 사이의 독특한 문화임을 깨닫는다. 그리고 감사하면서도, 너무나 모자른 나 자신이지만 스멀스멀 하나씩 뭔가를 기록하고 공유하고싶은 생각이 들기 시작한다.
어디에 글을 쓸 것인가? 간지나게 깃허브 블로그를 써볼까? 아니면 자체 서버로 블로그 홈페이지를 하나 꾸릴까? 뭔가 고수들만 쓰는 것 같은 medium 같은건 어떨까?
깃허브 블로그도 만들어 보았지만(이런 고민한 사람들은 대게 꼭 깃허브 레퍼토리에 github.io 가 있다.), 지금와서 생각해보면, 어디에다가 할까? 가 아니라 일단 하는게 중요하다는 생각이 든다.깃허브 블로그를 하면 지킬을 설치하고, 거기에 마크다운 문법에 맞춰 글을 써야한다.글 쓸 때마다 글 저장양식 맞춰야하고, 커밋찍어서 올려야 한다.수정할 때마다 커밋 찍어야하니, 프로필에 있는 내 커밋 판, 일명 모내기판이 의미없는 커밋들로 더러워질것만 같다.
자체 서버를 이용하여 블로깅 하면, 플랫폼 구애받지 않고 내 맘대로 할 수 있을 것 같긴한데, 호스팅 비용도 그렇거니와 데이터를 내가 잘 백업해둘지 영속적으로 내가 관리를 잘할지 모르겠다. 그리고 무엇보다 글을 쓰기 전에 홈페이지를 만들고, 에디터를 달아두고 하는게 너무 번거롭다.
그래서, 결국은 티스토리다. 폭넓은 자유도를 가지지는 못하지만 아무튼 글을 당장 빨리 쓸 수 있기 때문이다. 디자인, 에디터 그런거가 중요한게 아니라, 지금 당장 글 쓰고 퍼블리싱 하는게 제일 중요하다. 그리고 그 글이 잘 공유되는게 중요하다. 후에 더 필요한 우선순위가 떠오르면 그 때 바꿔도 무방하다고 생각든다!그래서 정했다. 티스토리.
아래에는 내가 이 티스토리 개발관련 블로깅을 하는 과정을 적어둔다.
1. 마크다운으로 글쓰기 - Typora

글에 코드가 들어가야 하는데, 티스토리에는 코드의 느낌을 살려주는 글쓰기 기능이 없다. 그래서 개발 관련 글을 쓸 때에는 마크다운으로 글을 써야겠다는 생각이 들었다. 찾아보��, Typora 라는 마크다운 에디터가 있는데 평이 좋더라.
Typora는 그냥 텍스트 에디턴데, 글을 쓰면 실시간으로 마크다운 문법으로 바꿔서 보여준다. 저장도 .md 뿐 아니라 html 도 지원한다. 때문에, 나는 Typora 를 이용하여 글을 쓰고, html로 내보내어 html 코드를 전체 복사하여, 티스토리 에디터의 html 로 저장하여 발행한다.
html 코드를 복사하여 티스토리 에디터에 붙여넣는게 조금 번거롭기는 한데, 뭐 이것말고는 깔끔하게 글을 보여줄만한 방법이 없다. 제일 좋은건 티스토리에도 마크다운 에디터가 내장되어있으면 좋겠는데, 아직까지 그런거는 없나보다.

[2020.02.08 수정]현재는 티스토리 글쓰기 에디터에 마크다운 에디터가 내장되어 있다.Typora 도 좋지만, 블로그에 바로 마크다운으로 글 써도 전혀 문제가 없다.

2. 발행된 글에 syntax highlight 넣기 - highlight.js

코드가 들어있는 글을 티스토리에 바로 올려서 발행하면, 코드가 이쁘게 나오기는 하는데 syntax highlight 가 안되어서 코드느낌이 안나온다.아래와 같이 밋밋한 느낌이다.
int main(){
  printf("hello world");
  return 0;
}
syntax highlight 가 뭐냐면, 아래 같이 프로그래밍 코드에서 문법을 색깔로 강조한 것을 말한다.
int main(){
  printf("hello world");
  return 0;
}이를 위해서 hightlight.js 를 설치해줘야하는데, 엄청나게 쉬우므로, 아 또 뭐 깔아야하네 하며 귀찮아 하지 말자... 금방한다.여기 설치를 위한 아주 훌륭한 포스팅이 있으므로, 이를 참고하여 따라가면 차암 쉽다.
주저리 주저리 말이 많았는데, 공부하며 글을 쓰고, 혹시나 모를 이를 위해 이를 공유해야겠다는 마음을 먹은 당신이라면. 얼른 티스토리 블로그를 켜 어떤 글이라도 하나 남겨보길 바란다.
